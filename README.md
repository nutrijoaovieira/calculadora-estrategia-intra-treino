# Calculadora de Estrategia Nutricional

Esta pasta contem a versao instalavel da calculadora.

## Como testar

Abra a pasta em um servidor local e acesse `index.html`. Em localhost, o navegador ja permite instalar o app e registrar o modo offline.

## Como publicar

Envie todos os arquivos desta pasta para um servico de hospedagem HTTPS, como Netlify, Vercel, Cloudflare Pages, GitHub Pages ou a hospedagem do seu site.

Depois de publicado, abra o link no celular:

- iPhone: Safari > Compartilhar > Adicionar a Tela de Inicio.
- Android: Chrome > menu > Instalar app ou Adicionar a tela inicial.

## Arquivos principais

- `index.html`: calculadora.
- `manifest.webmanifest`: nome, cores, icones e comportamento de app.
- `sw.js`: cache para abrir mais rapido e funcionar offline depois do primeiro acesso.
- `icons/`: icones do app.
- `assets/`: logo usada no PDF.
